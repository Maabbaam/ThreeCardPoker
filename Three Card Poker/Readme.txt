To run the program:
- Open threecardpoker.exe

To view the source code:
- Open file explorer
- Go to Downloads
- Enter the Three Card Poker directory
- Enter the src directory
- Open files in notepad by right-clicking on any Java source file and select Open With -> Notepad


To compile and run the program:
- Install Eclipse (https://www.eclipse.org/downloads/?)
- Within Eclipse go to File -> Import
- Select "General" then select "Existing Projects into Workspace"
- Press Next
- Select the Browse Button
- Go to Download
- Select the Three Card Poker directory
- Press "OK"
